# 🎵 React Music Player App

A modern and responsive music player built with **React**, **TailwindCSS**, and **Axios**. It fetches real-time music data using the **iTunes Search API**, allowing users to search, preview, and select songs interactively.

---

## 🚀 Features

- 🔍 **Search Music** by artist, track, or album
- 🎧 **Preview Audio** using native `<audio>` player
- 📀 **Display Album Art** with dynamic background
- ✨ **Smooth Hover Effects** with Tailwind transitions
- 🔁 **Selectable Tracks** to change the main view
- 🧠 **Built with Axios** to fetch data from the iTunes API

---

## 🛠️ Tech Stack

- **React** — UI library
- **Axios** — For API requests
- **TailwindCSS** — For styling and responsive design
- **iTunes Search API** — Music data source
